# abutalha.co.uk

Source for the personal portfolio website at [abutalha.co.uk](https://abutalha.co.uk).

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev -- --host 127.0.0.1 --port 5757
```

Build for production:

```bash
npm run build
```
